F.A.Q

What is Code Cubicle and what makes it unique?

The Code Cubicle is a groundbreaking hackathon jointly organized by Reskill and Geek Room. It provides a unique experience, redefining the boundaries of creativity and technology, transcending conventional hackathons.

When and where is Code Cubicle 2024 taking place?

Code Cubicle 2024 is an offline event scheduled for 19th-20th May. The venue is Microsoft Office Gurugram . It's a 36-hour hackathon that promises an immersive experience in tech innovation.

What is the prize pool for Code Cubicle, and how are the prizes distributed?

Code Cubicle boasts a prize pool worth INR 50,00,000+, with a cash prize worth INR 25,000+. Details about the prize distribution will be announced closer to the event.

What engaging activities can participants expect during Code Cubicle?

Code Cubicle features live project presentations, idea pitching, judging by esteemed panels, and a fun and games round with prizes to keep participants engaged throughout the hackathon.

What exclusive perks are provided to participants?

Participants at Code Cubicle enjoy exclusive perks, including free swags, networking opportunities with industry experts, goodies, accommodation, meals, and much more, enhancing their overall experience.

How does Code Cubicle encourage innovation and problem-solving?

Code Cubicle challenges participants to push their boundaries and foster inventive solutions to real-world issues. It celebrates the union of innovation and technology, providing a platform for participants to showcase their skills.

Who are the organizers of Code Cubicle, and what is their mission?

Code Cubicle is hosted by the collaborative efforts of Reskill and Geek Room. Their mission is to provide a platform for tech enthusiasts to showcase their skills and amplify their potential through an exciting blend of competition, learning, and groundbreaking innovation.

How can participants connect with like-minded individuals and industry experts during Code Cubicle?

Participants can connect with like-minded individuals and industry experts through networking sessions, project collaborations, and various interactive opportunities provided during Code Cubicle. It's a unique chance to learn, compete, and establish valuable connections.

Is there a code of conduct for participants?

Yes, participants are required to follow the code of conduct outlined at [Code of Conduct]. It ensures a positive and respectful environment for all attendees.

How can I stay updated on Code Cubicle announcements and details?

Stay updated on Code Cubicle announcements and details by regularly checking the official website. Additionally, follow their social media channels for the latest news and updates regarding the event.

How can I register for Code Cubicle 2024?

To register for Code Cubicle 2024, visit the official website and follow the registration instructions. The registration link on Devfolio can be found [here]. Complete the registration form and make sure to adhere to the specified deadlines.

What is the maximum team size allowed for Code Cubicle?

The maximum team size allowed for Code Cubicle is 4 members. Participants can form teams with a minimum of 1 member and a maximum of 4 members.

Can I participate as an individual, or is it mandatory to be part of a team?

Code Cubicle allows both individual and team participation. You can register as an individual and participate solo, or you can form a team with up to 3 other members.

Are there any specific requirements for the projects that participants can work on?

Participants in Code Cubicle have the flexibility to choose the type of projects they want to work on. There are no specific requirements, allowing participants to showcase their creativity and skills by working on innovative solutions to real-world issues.

What should I bring to Code Cubicle, and what will be provided by the organizers?

Participants are encouraged to bring their laptops and any specific tools or hardware they may need for coding. Organizers will provide the necessary infrastructure, but participants should ensure they have everything they need for a successful hacking experience.
